Enigma

Enigma is the name of the german cipher machine used in World War II. The word aenigma is being derived itself from the ancient greek word a√≠nigma. Enigma means something or someone puzzling, mysterious or inexplicable.

Firmware

In computer science, firmware is software that enables low-level control of the hardware of computer devices. For a relatively simple device, the firmware can perform all control, monitoring and data processing functions. For a more complex device, firmware can provide both relatively simple control and also hardware control for high-level software, such as an operating system. Firmware is found in a variety of computing devices such as personal computers, computer peripherals, telecommunication devices, home entertainment, and vehicles. Firmware is stored mostly in programmable memory such as EPROM or EEPROM as well as flash memory. Programmable firmware memory can be reprogrammed through a process sometimes referred to as flashing. Common reasons for firmware changes are to fix bugs and add features.

Wayland

X11, Xorg or the X Window System lets a graphical desktop environment display and control windows under Linux. In this context Wayland is new a communication protocol that specifies the communication between a display server and its clients. The aim of Wayland is replacing the X Window System with a modern, faster, more secure and simpler windowing system.
